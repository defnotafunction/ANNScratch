# ANNScratch
- As the title states, this project's goal is to replicate an artifical neural network from scratch with numpy as an exception (other libraries are used for testing and graphing).

## Features
- Perceptron: The perceptron is the simplest building block of a neural network. It takes in in inputs, multiplies it with weights, adds a bias, then its output is determined by an activation function.
- ANN: The artificial neural network will be composed of multiple Perceptrons. The layers of Perceptrons will enable it to grasp complex data.
- Loss Functions: The project also includes loss functions like MSE.
- Activation Functions: The project also includes activation functions like ReLU.

## Technologies Used
- NumPy
- Scikit-learn
- Matplotlib
